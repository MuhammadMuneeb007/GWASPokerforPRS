#===============================================
# README for GWAS catalog data for DAYA et.al paper
#===============================================

There are three data dictionary files

#-----------------------------------------------
1. META_analysis_summaryStatistics_datadictionary.xlsx
#-----------------------------------------------
This file contains the data dictionary for files in the "META_analysis_summaryStatistics" directory.
The "META_analysis_summaryStatistics" directory contains files with meta-analysis summary statistics for all cohorts generated using MR-MEGA.   

  File extension: *.txt.gz
  
#-----------------------------------------------
2. AllCohorts_associationanalysis_summaryStatistics_datadictionary.xlsx
#-----------------------------------------------
This file contains the data dictionary for all files in the "AllCohorts_associationanalysis_summaryStatistics" directory.  

The "AllCohorts_associationanalysis_summaryStatistics" directory contains files with single cohort summary statistics generated using GENESIS.

  File extension: *.txt.gz
  
#-----------------------------------------------
3. SAPPHIRE_associationanalysis_summaryStatistics_datadictionary.xlsx
#-----------------------------------------------
This file contains the daata dictionary for all files in the "SAPPHIRE_associationanalysis_summaryStatistics" directory.  

The "SAPPHIRE_associationanalysis_summaryStatistics" directory contains files for the SAPPHIRE cohort generated using PLINK.

  File extension: *.txt.gz


#===============================================#
